# Task3-Retail-Store-Sales-DeepDive-Analysis
Deep Dive Analysis and Interactive Dashboard on Retail Sales Data using Python and Power BI


## 📌 Project Overview
This project focuses on performing a deep-dive analysis of retail transaction data and building an interactive dashboard to uncover business insights.

## 🎯 Objective
To analyze customer behavior, revenue trends, and transaction patterns, and to develop an interactive dashboard for data-driven decision-making.

---

## 📊 Key Performance Indicators (KPIs)

- **Total Revenue** = SUM(total_spent)
- **Average Order Value (AOV)** = Total Revenue / Total Transactions
- **Total Customers** = DISTINCT COUNT(customer_id)
- **Average Quantity per Order**
- **Discount Impact on Spending**

---

## 🔍 Deep-Dive Analysis: Customer Segmentation

Customers were segmented based on total spending:

- **High Value** (> 5000)
- **Medium Value** (2000 – 5000)
- **Low Value** (< 2000)

This segmentation helps identify high-value customers and target growth opportunities.

---

## 📈 Dashboard Features

The interactive dashboard includes:

- KPI Cards (Revenue, Customers, AOV)
- Monthly Revenue Trend
- Revenue by Category
- Customer Segmentation Analysis
- Payment Method Distribution
- Filters (Category, Location, Month)

---

## 🧠 Key Insights

- High-value customers contribute the majority of revenue
- Medium-value customers represent growth potential
- Discounts influence customer spending patterns
- Certain payment methods dominate revenue contribution
- Seasonal trends observed in monthly sales

---

## 🛠️ Tools Used

- Python (Pandas, Matplotlib, Seaborn)
- Power BI


---

## 📌 Conclusion

This project demonstrates how raw retail data can be transformed into actionable insights using data analysis and visualization techniques.
